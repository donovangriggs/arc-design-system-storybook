# Design Tokens

Design tokens are the visual foundation of the ArcTech Design System. They are delivered as CSS custom properties (prefixed with `--arc-`) and can be consumed in any framework or plain CSS.

```css
@import '@arctech/tokens/css/tokens.css';
```

## Usage in CSS

```css
.my-component {
  color: var(--arc-color-primary-500);
  padding: var(--arc-spacing-4);
  font-size: var(--arc-font-size-base);
  font-family: var(--arc-font-family-sans);
  border-radius: var(--arc-border-radius-md);
  box-shadow: var(--arc-shadow-md);
  transition: all var(--arc-transition-normal);
}
```
