# ArcTech Design System

A cross-framework design system built with Web Components. Use the same components in React, Angular, Vue, and Svelte.

---

## Quick Stats

| | |
|---|---|
| **Components** | 9 production-ready UI components |
| **Framework Wrappers** | React, Angular, Vue, Svelte |
| **Tokens** | Tailwind-compatible design tokens via CSS custom properties |
| **Theming** | Light and dark mode with system preference detection |

---

## Packages

| Package | Description |
|---|---|
| `@arctech/core` | Web Components (Stencil-based) |
| `@arctech/react` | React wrapper components |
| `@arctech/angular` | Angular wrapper components |
| `@arctech/vue` | Vue wrapper components |
| `@arctech/svelte` | Svelte wrapper components |
| `@arctech/tokens` | Design tokens (CSS, JS, JSON) |

---

## Components

- **ArcButton** -- Buttons with variants, sizes, loading states, and icons
- **ArcInput** -- Text input fields with validation and labels
- **ArcCard** -- Content containers with header, body, and footer slots
- **ArcAvatar** -- User avatars with image, initials, and status indicators
- **ArcBadge** -- Status badges and labels
- **ArcAlert** -- Notification banners for success, warning, error, and info
- **ArcTooltip** -- Contextual tooltips on hover or focus
- **ArcIcon** -- SVG icon rendering
- **ArcModal** -- Dialog overlays with configurable sizes
